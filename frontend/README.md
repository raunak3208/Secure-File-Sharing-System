# Secure File Sharing - Frontend

React.js frontend for the Secure File Sharing application.

## Structure

```
frontend/
├── src/
│   ├── components/      # Reusable React components (.jsx)
│   ├── pages/           # Page components (.jsx)
│   ├── services/        # API service calls (.js)
│   ├── hooks/           # Custom React hooks (.js)
│   ├── utils/           # Utility functions (.js)
│   └── styles/          # CSS and styling
├── public/              # Static assets
└── package.json
```

## Components
- Login/Register forms
- Dashboard with file management
- File sharing modal
- Secure PDF viewer
- File upload component

## Pages
- Home page
- Login page
- Register page
- Dashboard page
- Shared file viewer page

## Services
- Supabase authentication
- File upload/download
- File access control
- Device binding
- Security logging
